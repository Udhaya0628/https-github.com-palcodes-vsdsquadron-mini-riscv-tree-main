# Task 3 - UART Driver Demo

## Description
Implemented UART communication on VSDSquadron Mini using CH32V003.

## Features
- UART initialization
- Send and receive characters
- LED ON/OFF control
- UART command interface

## Commands
- 1 -> LED ON
- 0 -> LED OFF
- t -> LED TOGGLE
- h -> HELP MENU
